# PyCloud
Nextcloud alternative made with python, js, html, css. Easy to run, fast and currently in development.
# Key features
### File Storage
### Notes
### AI Chat (Ollama)
### General Group Chat
### Admin Settings
### Password reset using smtp
# Upcoming features
